# TruSpace Static Website

A pure HTML/CSS version of the TruSpace website

## Features

- **No JavaScript required** - Pure HTML5 and CSS3
- **No external fonts** - Uses system font stack for fast loading
- **Fully responsive** - Works on mobile, tablet, and desktop
- **Modern CSS** - Uses CSS Grid, Flexbox, Custom Properties
- **Accessible** - ARIA labels, semantic HTML, keyboard navigation
- **Print-friendly** - Optimized print stylesheet included
- **SEO-ready** - All meta tags, Open Graph, Twitter Cards, and structured data preserved
